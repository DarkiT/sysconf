# Sysconf - Go Configuration Management Library

## Quick Start

```bash
go get github.com/darkit/sysconf
```

```go
package main

import (
    "fmt"
    "log"

    "github.com/darkit/sysconf"
)

func main() {
    cfg, err := sysconf.New(
        sysconf.WithPath("configs"),
        sysconf.WithName("app"),
        sysconf.WithMode("yaml"),
    )
    if err != nil {
        log.Fatal(err)
    }

    host := cfg.GetString("database.host", "localhost")
    port := cfg.GetInt("database.port", 5432)

    fmt.Printf("Database: %s:%d\n", host, port)
}
```

## Overview

Sysconf is a high-performance, thread-safe Go configuration management library designed for enterprise applications. It uses atomic storage technology and a smart validation system to provide reliable concurrent access support.

### Key Features

- **⚡ High Performance**: Microsecond-level config reads, millisecond-level writes
- **🔒 Thread-Safe**: Based on `atomic.Value` + `sync.RWMutex`
- **📊 Concurrent Support**: Strictly tested for high-concurrency scenarios
- **💾 Smart Caching**: Atomic storage with cache optimization
- **🛡️ Smart Validation**: Field-level validation with 30+ built-in rules
- **🔐 Advanced Encryption**: ChaCha20-Poly1305 authenticated encryption
- **🌐 Environment Variables**: Smart case matching
- **🔄 Hot Reload**: Debounced file watching
- **🎯 Viper Compatible**: Drop-in replacement for spf13/viper

### Basic Configuration

```go
import "github.com/darkit/sysconf"

cfg, err := sysconf.New(
    sysconf.WithPath("configs"),           // Config directory
    sysconf.WithName("app"),               // Config file name
    sysconf.WithMode("yaml"),              // Format: yaml, json, toml
    sysconf.WithContent(defaultConfig),    // Default config content

```

### Type-Safe Reading

```go
// Basic types
host := cfg.GetString("database.host", "localhost")
port := cfg.GetInt("database.port", 5432)
debug := cfg.GetBool("app.debug", false)
timeout := cfg.GetDuration("database.timeout")

// Slices
features := cfg.GetStringSlice("server.features")
ports := cfg.GetIntSlice("server.ports")

// Maps
options := cfg.GetStringMap("database.options")

// Generic API (Go 1.24+)
host := sysconf.GetAs[string](cfg, "database.host", "localhost")
port := sysconf.GetAs[int](cfg, "database.port", 5432)
apiKey := sysconf.MustGetAs[string](cfg, "api.secret_key") // Panics if missing
```

### Writing Configuration

```go
// Single key
err := cfg.Set("database.host", "new-host")

// Batch update (atomic)
err := cfg.SetMultiple(map[string]any{
    "database.host": "new-host",
    "database.port": 5433,
    "app.debug": true,
})
```

### Validation System

```go
import "github.com/darkit/sysconf/validation"

cfg, err := sysconf.New(
    sysconf.WithValidators(
        validation.NewDatabaseValidator(),
        validation.NewWebServerValidator(),
    ),
)

// Invalid values are automatically rejected
cfg.Set("server.port", 8080)     // ✅ Valid

```

## Architecture

### Core Design

```
┌─────────────────────────────────────────────────────────┐
│                      Config Core                        │
│  ┌─────────────────────────────────────────────────┐   │
│  │  data atomic.Value (map[string]any)            │   │
│  │  mu sync.RWMutex                                │   │
│  │  cache *Cache                                   │   │
│  │  validators []ConfigValidator                   │   │
│  │  crypto ConfigCrypto                            │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### Concurrency Model

- **Read Path**: Lock-free via `atomic.Value.Load()` (~100ns)
- **Write Path**: Protected by `sync.RWMutex` (~1-5ms)
- **Validation**: Lock-free reads of validator slice

### Performance

| Operation | Latency |
| --------- | ------- |
| GetString | ~100ns  |
| GetInt    | ~200ns  |
| GetAs[T]  | ~300ns  |
| Set       | ~1-5ms  |

## Migration from Viper

```go
// Before (viper)
import "github.com/spf13/viper"
v := viper.New()
v.SetConfigName("config")
v.ReadInConfig()

// After (sysconf)
import "github.com/darkit/sysconf"
cfg, _ := sysconf.New(
    sysconf.WithName("config"),
)

// API is compatible
host := cfg.GetString("database.host")
```
