# Sysconf - Go 配置管理库（Go 1.25+ 版本）

## 快速开始

```bash
go get github.com/darkit/sysconf
```

```go
package main

import (
	"fmt"
	"log"

	"github.com/darkit/sysconf"
)

func main() {
	cfg, err := sysconf.New(
		sysconf.WithPath("configs"),
		sysconf.WithName("app"),
		sysconf.WithMode("yaml"),
	)
	if err != nil {
		log.Fatal(err)
	}

	// 类型安全的泛型 API（Go 1.25+ 推荐）
	host := cfg.GetAs[string]("database.host", "localhost")
	port := cfg.GetAs[int]("database.port", 5432)

	fmt.Printf("数据库: %s:%d\n", host, port)
}
```

## 核心 API

### 基础配置

```go
import "github.com/darkit/sysconf"

cfg, err := sysconf.New(
	sysconf.WithPath("configs"),           // 配置文件目录
	sysconf.WithName("app"),               // 配置文件名
	sysconf.WithMode("yaml"),              // 格式：yaml, json, toml
	sysconf.WithContent(defaultConfig),    // 默认配置内容
)
```

### 类型安全读取（泛型 API）

```go
// 基础类型 - Go 1.24+ 的现代泛型 API
host := cfg.GetAs[string]("database.host", "localhost")
port := cfg.GetAs[int]("database.port", 5432)
debug := cfg.GetAs[bool]("app.debug", false)
timeout := cfg.GetAs[time.Duration]("database.timeout", 30*time.Second)

// 切片
features := cfg.GetAs[[]string]("server.features")
ports := cfg.GetAs[[]int]("server.ports")

// 映射
options := cfg.GetAs[map[string]string]("database.options")
```

### 强制读取（新增）— MustGetAs[T]

当配置项是**必填**且缺失时应该立即 panic，适合启动时的关键配置：

```go
// 如果不存在或转换失败，立即 panic
apiKey := sysconf.MustGetAs[string](cfg, "api.secret_key")
dbURL := sysconf.MustGetAs[string](cfg, "database.url")

// 与 GetAs 的区别：GetAs 有默认值；MustGetAs 无妥协
// GetAs[string](cfg, "key", "default") → "default" if missing
// MustGetAs[string](cfg, "key") → panic if missing
```

### 写入配置

```go
// 单键写入
err := cfg.Set("database.host", "new-host")

// 批量更新（原子操作）
err := cfg.SetMultiple(map[string]any{
	"database.host": "new-host",
	"database.port": 5433,
	"app.debug": true,
})
```

### 环境变量

cfg, err := sysconf.New(

### 并发模型

- **读路径**：无锁 `atomic.Value.Load()` (~100ns)
- **写路径**：`sync.RWMutex` 保护 (~1-5ms)
- **协程管理**：Go 1.25+ `errgroup.Go()` 代替手动 `Add(1) + defer Done()`

## 常见用法示例

### 企业级应用启动

```go
// 集成智能验证器
cfg, err := sysconf.New(
	sysconf.WithContent(defaultConfig),
	sysconf.WithValidators(
		validation.NewDatabaseValidator(),
		validation.NewWebServerValidator(),
		validation.NewRedisValidator(),
	),
	sysconf.WithEncryption(os.Getenv("CONFIG_KEY")),
)
if err != nil {
	panic(fmt.Sprintf("配置初始化失败: %v", err))
}

// 强制读取关键配置（缺失时 panic）
dbURL := sysconf.MustGetAs[string](cfg, "database.url")
apiKey := sysconf.MustGetAs[string](cfg, "api.key")

// 可选配置用 GetAs 并设默认值
port := cfg.GetAs[int]("server.port", 8080)
```

### 高并发场景

```go
// GetAs 是无锁读取，安全用于高并发
var wg sync.WaitGroup
for i := 0; i < 1000; i++ {
	wg.Add(1)
	go func() {
		defer wg.Done()
		_ = cfg.GetAs[string]("database.host")
	}()
}
wg.Wait()
```
